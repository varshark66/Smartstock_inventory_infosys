"Transforming traditional inventory management into a smart, efficient, and data-driven system."

👥 Team Project

This project was developed as part of a team collaboration.
I contributed to the design, development, and implementation of key features in the system.

📖 Project Overview

SmartStock is a web-based inventory management system designed to help retail businesses efficiently manage stock, track sales, and automate alerts.

It replaces manual inventory tracking with a real-time, automated, and user-friendly solution, suitable for both small and large-scale businesses.

🎯 Key Features
📦 Real-time inventory tracking
🔔 Automated low-stock alerts
👤 Role-based access (Admin & Employee)
📊 Sales and transaction management
📈 Reports and analytics dashboard
💱 Multi-currency support
✉️ Email notifications for users
🔐 Secure authentication (JWT)
🧾 Complete transaction history
🤖 Smart assistant for queries
🏗️ System Architecture
🔄 Architecture Flow

Frontend → Backend → Database → Alerts System

⚙️ Components
💻 Frontend (React) – Interactive UI
🔧 Backend (Flask APIs) – Business logic & authentication
🗄️ Database (MongoDB) – Stores all data
📧 Alerts System (SMTP) – Sends notifications
